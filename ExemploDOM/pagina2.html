DOM

O DOM (Document Object Model) representa a estrutura da página HTML como uma árvore de elementos. Com JavaScript, é possível acessar, modificar e interagir com esses elementos.

💻

Variáveis

Variáveis armazenam dados na memória. Em JavaScript, usamos let, const e var para guardar e manipular informações durante a execução do código.

📊

Tipos de Dados

Definem o tipo de valor armazenado. Os principais são string, number e boolean, além de array e object para estruturas mais complexas.

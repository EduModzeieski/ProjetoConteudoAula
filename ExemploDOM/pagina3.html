Eventos em JavaScript!

Aprenda como os eventos permitem que o usuário interaja com a página, como nos cliques das peças!

🖱️

Eventos de Clique

O evento onclick é usado para detectar quando o usuário clica em um elemento. No projeto, cada peça tem onclick="ativarPeca(pecas[0])" para ativar a peça correspondente.

⏱️

Eventos Temporizados

Usamos setTimeout para criar animações, como a classe 'clicando' que dura 200ms. Isso adiciona feedback visual aos cliques sem bloquear a interação.

🔗

Redirecionamento

Após ativar uma peça, um botão aparece para redirecionar para a próxima página usando window.location.href. Isso cria uma experiência sequencial de aprendizado.

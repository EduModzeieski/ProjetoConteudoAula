Loops e Iteração!

Veja como os loops são usados no projeto para processar arrays de objetos, como o array pecas[]!

🔄

Loop forEach

O forEach é usado para executar uma função em cada elemento de um array. No projeto, pecas.forEach(config => { ... }) é usado em desativarTodasPecas para resetar todas as peças antes de ativar uma nova.

❓

Loop while

O while executa enquanto uma condição for verdadeira. Embora não usado diretamente no projeto, é útil para cenários onde o número de interações não é conhecido antecipadamente.

🔁

Interação sobre Arrays

Loops permitem processar coleções de dados. O array pecas[] é iterado para aplicar lógica a cada peça, demonstrando como loops facilitam a manipulação de múltiplos elementos de forma eficiente.

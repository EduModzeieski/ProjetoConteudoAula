Objetos e Arrays!

Explore como objetos e arrays estruturam os dados das peças no quebra-cabeça interativo!

📦

Criando Objetos

Objetos são criados usando chaves {}. No projeto, cada peça é representada por um objeto no array pecas[], contendo propriedades como numero, corAtiva, emojiAtivo e métodos para controlar seu estado.

🔑

Propriedades e Métodos

Propriedades armazenam valores, como pecaConfig.corAtiva. Acesse-os usando notação de ponto. No código, pecaConfig.ativo controla se a peça está ativa, e pecaConfig.textoAtivo define a mensagem exibida.

🏗️

Arrays de Objetos

O array pecas[] contém objetos para cada peça, permitindo acesso fácil via índice: pecas[0] para a primeira peça. Isso organiza dados complexos e facilita a iteração e manipulação no JavaScript.

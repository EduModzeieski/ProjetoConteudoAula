Programação Website I

Estado e Progresso!

Veja como o projeto guarda o estado das peças e mostra o progresso do usuário ao interagir com o quebra-cabeça.

📊

Contagem de Progresso

O código atualiza o contador de peças ativadas e preenche a barra de progresso. Isso ajuda o usuário a ver o quanto já avançou no desafio.

💾

Session Storage

O estado de cada peça é salvo no sessionStorage para manter a informação durante a sessão. Assim, a ativação é lembrada enquanto a aba do navegador estiver aberta.

✅

Experiência Interativa

O uso de estados e armazenamento permite criar uma experiência interativa mais consistente, com peças ativadas, status visíveis e retorno de informações para o usuário.
